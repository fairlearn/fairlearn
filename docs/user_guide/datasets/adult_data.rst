.. _adult_data:

Adult Census Dataset
--------------------

This section contains datasets created by
Ding et al. :footcite:`ding2021retiring`.
The authors intend for their dataset to replace the
UCI Adult dataset :footcite:`kohavi1996adult`,
which is a popular benchmark dataset for comparing algorithmic interventions 
with respect to fairness metrics.
